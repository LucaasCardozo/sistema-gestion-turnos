<h1>Sistema de Gestión de Turnos</h1>

<p>
Aplicación web para la gestión genérica de turnos, que permite administrar clientes, empleados, sucursales y servicios de manera eficiente.
</p>

<h2>Demo</h2>
<p>
<a href="https://drive.google.com/file/d/1CX-Hr-j7I-ZoxCaizOEOn7_mQvrGdS4n/view?usp=drive_link">Ver demostración general del sistema</a>
</p>
<p>
<a href="https://drive.google.com/drive/folders/1V3AxVnhU5V6A9YANPLzf4IH6dqTN-66j?usp=sharing">Ver mejoras y funcionalidades agregadas</a>
</p>

<h2>Funcionalidades</h2>
<ul>
  <li>Gestión de turnos (alta, baja y modificación)</li>
  <li>Administración de clientes y empleados</li>
  <li>Gestión de sucursales y servicios</li>
  <li>Configuración de disponibilidad y días de atención</li>
  <li>Sistema de roles (administrador, empleado y cliente)</li>
  <li>Registro y autenticación de usuarios</li>
</ul>

<h2>Tecnologías utilizadas</h2>
<ul>
  <li>Java</li>
  <li>Spring Boot</li>
  <li>MySQL</li>
  <li>HTML, CSS, JavaScript</li>
  <li>Bootstrap</li>
  <li>Git y GitHub</li>
</ul>

<h2>Trabajo en equipo</h2>
<p>
Proyecto desarrollado en equipo (4 integrantes) en el marco de la materia <strong>Orientación a Objetos 2</strong>.
</p>

<h2>Ejecutar el proyecto</h2>
<ol>
  <li>Clonar el repositorio</li>
  <li>Configurar una base de datos MySQL</li>
  <li>Configurar las credenciales en el archivo <strong>application.yml</strong></li>
  <li>Ejecutar la aplicación desde el entorno de desarrollo</li>
</ol>
